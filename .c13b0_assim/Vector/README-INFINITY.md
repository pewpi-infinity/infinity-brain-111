# Infinity App Repo Rules

1. **Code Quality**: All contributions must adhere to the coding guidelines of the project.
2. **Issue Tracking**: Any bugs or feature requests should be reported in the issue tracker.
3. **Pull Requests**: All changes must be made via a pull request. 
4. **Documentation**: All new features must be documented.
5. **Testing**: New features and bug fixes must include relevant tests.
6. **Respect the Community**: Ensure a positive and respectful community atmosphere.

For more information, refer to our [Contributing Guide](link-to-guide).
